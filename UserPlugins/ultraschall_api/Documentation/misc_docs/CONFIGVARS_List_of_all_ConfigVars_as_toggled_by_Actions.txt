#ConfigVars triggered by Reaper's actions in the Main-Section.
#
#Groups of three are toggle actions and therefore you can see, which is the dif between three toggles.
#The dif between second and third should be usually the dif-value you are looking for.
#
#Groups of only one are actions, who only set a config-var once, but do not toggle it.
#Some actions do not have a name but still trigger config-vars. It is unknown, why.
#
#This list is probably not comprehensive, keep this in mind.
